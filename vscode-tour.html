What is VS Code?
Visual Studio Code (VS Code) is a free code editor made by Microsoft. Think of it like Microsoft Word — but instead of writing essays, you write code. It comes packed with powerful tools that help you write, organize, run, and fix code. It works with almost every programming language and is used by millions of developers worldwide.

🖥️ Integrated Terminal Ctrl+`
user@computer:~/my-project$ python3 app.py
Hello, World!
user@computer:~/my-project$ ▌
A built-in command line inside VS Code. You use it to run your programs, install packages, and talk to your computer directly. No need to open a separate terminal app.
Press Ctrl+` (backtick key, top-left of keyboard) to open it

When your code has a bug (an error), the debugger helps you find it. You can pause your program at any line (called a breakpoint) and inspect what every variable's value is at that exact moment.
Click the red dot (●) area to the left of a line number to set a breakpoint
